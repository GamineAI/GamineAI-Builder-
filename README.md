## 🚀 Features of GamineAI Builder

### 🎮 Game Creation
- Generate full games using AI: **Tetris, Pong, Snake, Flappy Bird, and custom games**.  
- Customize **game type, theme, difficulty, size, visual style, and gameplay**.  
- Download, share, and play games instantly.

### 🛠 AI Templates
- **NPC Dialogue System** – Dynamic characters with personality and memory.  
- **Level Generation** – Procedural level creation with AI assistance.  
- **Combat AI** – Intelligent enemy behavior and combat mechanics.  
- **Quest System** – Dynamic quest generation and management.  

### 🎨 AI Theme Generator
- Generate **visual themes** for games: colors, styles, and full CSS.  
- Customize **visual presets, gameplay style, audio style, and color palettes**.  
- Preview, apply, and save themes directly.

### 🎵 Sound Effects Generator
- Generate sound effects: **explosions, jumps, hits, UI, music, ambient sounds**.  
- Customize style: Retro, Modern, Cartoon, Orchestral, Electronic.  
- Manage generated sounds in a **sound library**.

### 💻 Code Editor
- Edit **HTML, CSS, JavaScript** of generated games.  
- Live preview with **profiler and auto-refresh**.  
- Export code or create a Gist directly.

### ⚡ Game Management
- **Game History** – Track generated games, favorites, and quick access.  
- **Backup & Restore** – Automatic and manual backups to prevent data loss.  
- **Statistics Dashboard** – Track game generation stats.  
- **Console Logging** – Debug info, warnings, and errors.
